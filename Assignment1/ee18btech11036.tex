\section{Problem}
(7.1) The command
\begin{lstlisting}
output_signal = signal.lfilter(b, a, input_signal)
\end{lstlisting}
in Problem (2.3) is executed through the following difference equation
\begin{equation}
\label{eq:iir_filter_gen}
 \sum _{m=0}^{M}a\brak{m}y\brak{n-m}=\sum _{k=0}^{N}b\brak{k}x\brak{n-k}
\end{equation}
where the input signal is $x(n)$ and the output signal is $y(n)$ with initial values all 0. Replace
\textbf{signal.filtfilt} with your own routine and verify.
\section{Solution}
This problem can be solved by using $Z$ transfrom on the difference equation and then applying the properties of $Z$ transform.
Let take $X(z)$ and $Y(z)$ as the z-transforms of $x(n)$ and $y(n)$.
\newline
Using the time shifting property, 
  \begin{align}
      {\mathcal {Z}}\{x(n-k)\} = z^{-k}X(z) \\
      {\mathcal {Z}}\{y(n-m)\} = z^{-m}Y(z)
  \end{align}
\newline
We get:
\begin{align}
     Y\brak{z} \sum _{m=0}^{M}a\brak{m} z^{-m} = X\brak{z} \sum _{k=0}^{N}b\brak{k}z^{-k}
\end{align}
\begin{align}
\label{eq:Hz}
    H\brak{z} = \frac{Y\brak{z}}{X\brak{z}} = \frac{\sum _{k=0}^{N}b\brak{k} z^{-k}}{\sum _{m=0}^{M}a\brak{m}z^{-m}}
\end{align}    
Now to obtain DFT from Z transform, we evaluate \eqref{eq:Hz} at $z = e^{j \omega}$ where $\omega = 2 \pi k/N$
\newline
Hence $H(k)$ :
\begin{equation}
H\brak{k} = H\brak{z = e^{j 2 \pi k/N}}.
\end{equation}
$X(k)$ is calculated from our input signal $x(n)$ using \textbf{np.fft.fft}.
\newline
Therefore $Y(k)$:
\begin{align}
    Y\brak{k} = H\brak{k}X\brak{k}
\end{align}
Since this is in frequency domain, $y(n)$ is found from $Y(K)$ using \textbf{np.fft.ifft} command.
